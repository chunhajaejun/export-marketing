기능 설명

이 화면에서

각 매체의 일별 광고 소진액을 입력합니다. 텍스트 붙여넣기 또는 직접 입력 모드를 선택할 수 있습니다.

이력 확인

입력된 소진액은 즉시 우측 이력에 반영되며, 콜량 대비 단가(전체/유효)가 자동 계산됩니다.

미입력 알림

아직 소진액이 입력되지 않은 매체는 "미입력"으로 표시되어 누락을 방지합니다.
