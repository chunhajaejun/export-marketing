텍스트 붙여넣기

카카오톡이나 디비마스터에서 받은 보고 텍스트를 그대로 붙여넣기하면 시스템이 자동 파싱하여 날짜, 매체, 건수를 추출합니다.

직접 입력

날짜와 매체를 선택하고 수출/중고/폐차/부재/무효 등 세부 건수를 직접 숫자로 입력합니다. 합계가 자동 계산됩니다.
