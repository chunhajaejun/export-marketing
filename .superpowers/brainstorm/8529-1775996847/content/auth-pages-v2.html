📋 인증 플로우 요약

1. 회원가입 → 이름, 이메일, 비밀번호, 소속(선택) 입력 → 가입 신청
2. 승인 대기 → 로그인하면 대기 화면 표시 (접근 차단)
3. 관리자 승인 → 사용자 관리 페이지에서 승인 + 역할 부여
4. 로그인 → 역할에 맞는 화면 접근 가능
비번 찾기 → 이메일로 재설정 링크 발송 (Supabase 기본 기능)
